compactOverlaySchedule ∷ OverlaySchedule crypto → Map (OBftSlot crypto) (NonEmpty SlotNo) Source #

Convert the overlay schedule to a representation that is more compact when serialised to a bytestring, but less efficient for lookups.

Each genesis key hash will only be stored once, instead of each time it is assigned to a slot.
